# ZOMFG!!! ZOMBIEEEEES!!!!!11!oneone!1
Orthogonal view zombie killing game.
Made with Ruby/Gosu in a week.
Winner of the HBGames Gosu Contest!

Controls:
Enter - Select option in menu
Esc - Exit the game
Arrows - Move around
Spacebar - Shoot, keep pressed for autofire
R - Reload
